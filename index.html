SQL Script

Get Latest Inventory Receipts

Pulls the last 50 item receipts including vendor details.

SELECT 
    r.Receipt_ID, 
    r.Item_Code, 
    i.Item_Name, 
    r.Quantity, 
    r.Receipt_Date
FROM tbl_Receipts r
JOIN tbl_ItemMaster i ON r.Item_Code = i.Item_Code
ORDER BY r.Receipt_Date DESC
LIMIT 50;
