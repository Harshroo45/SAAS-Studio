SQL Query Library

Essential CRUD operations, data extraction, and database modifications.

DML - Read

Get Latest Inventory Receipts

Safe

Pulls the last 50 item receipts including vendor details.

VIESL_LIVE 
SELECT 
    r.Receipt_ID, 
    r.Item_Code, 
    i.Item_Name, 
    r.Quantity, 
    r.Receipt_Date
FROM tbl_Receipts r
JOIN tbl_ItemMaster i ON r.Item_Code = i.Item_Code
ORDER BY r.Receipt_Date DESC
LIMIT 50;
DML - Update

Correct Stock Quantity

Caution

Manually overrides the stock level. Always ensure WHERE clause is active.

VIESL_LIVE 
BEGIN TRAN;

UPDATE tbl_ItemMaster
SET Current_Stock = @CorrectValue,
    Last_Modified = GETDATE()
WHERE Item_Code = 'ITM-10024';

-- COMMIT TRAN; -- Uncomment after verification
DML - Delete

Remove Stalled Gate Entry

High Risk

Deletes an abandoned gate entry. Fails if foreign keys exist in POs.

VIESL_LIVE 
DELETE FROM tbl_GateEntry
WHERE Entry_ID = @GateID
AND Status = 'STALLED';
